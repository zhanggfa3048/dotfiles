# dotfiles

个人配置文件备份，用于快速恢复开发环境。

## 📁 目录结构

```
dotfiles/
├── openclaw/           # OpenClaw 配置
├── nas/                # NAS Docker 配置
├── shell/              # Shell 配置
└── scripts/            # 常用脚本
```

## 🚀 包含内容

- 🤖 **OpenClaw** - AI 助手配置
- 🗄️ **NAS** - Docker Compose 配置文件
- 🐚 **Shell** - zsh/bash 配置
- 📜 **Scripts** - 常用自动化脚本

## 📥 使用方法

```bash
git clone https://github.com/zhanggfa3048/dotfiles.git
cd dotfiles
# 根据需要复制配置文件到对应位置
```

## ⚠️ 注意事项

敏感信息（API keys、密码等）已脱敏，使用前需手动填入。
